================================================================================
  ⚡ SUPER QUICK START (15 MENIT)
================================================================================

1. Push ke GitHub
   ---
   cd C:\laragon\www\kartu_stok_barang
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR_USERNAME/kartu-stok-barang.git
   git branch -M main
   git push -u origin main

2. Buat account Railway.app
   Buka: https://railway.app
   Sign up dengan GitHub

3. Create Project di Railway
   Dashboard → New Project → Select repo → Deploy

4. Setup MySQL Database
   Add MySQL service → Copy credentials ke Variables

5. Setup Telegram
   php artisan migrate --force
   php artisan telegram:set-webhook

6. Test Bot
   Buka Telegram → kirim /start
   Bot harus reply! ✅

SELESAI! Bot jalan 24/7! 🚀
